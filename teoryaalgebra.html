Теорія Алгебри

1. Основні поняття алгебри

Теорія: Алгебра — це розділ математики, який вивчає властивості чисел та операцій над ними, включаючи додавання, віднімання, множення та ділення. Алгебра використовує символи (змінні) для представлення чисел у рівняннях та виразах.

Приклад 1: x + 5 = 10. Тоді x = 5

Приклад 2: 3a - 2 = 7. Тоді a = 3

Приклад 3: 2(x + 3) = 10. Тоді x = 2.

Приклад 4: 4y = 20. Тоді y = 5

Приклад 5: 2x - 7 = 13. Тоді x = 10

На Головну

2. Розв'язування лінійних рівнянь

Теорія: Лінійне рівняння — це рівняння першого ступеня, в якому невідома з'являється лише в першій степені. Для розв'язування таких рівнянь зазвичай використовуються прості операції додавання, віднімання, множення та ділення.

Приклад 1: x + 4 = 10, x = 10 - 4, x = 6

Приклад 2: 2x - 5 = 9, 2x = 9 + 5, x = 14 / 2, x = 7

Приклад 3: 3x + 8 = 17, 3x = 17 - 8, x = 9 / 3, x = 3

Приклад 4: 4x - 3 = 13, 4x = 13 + 3, x = 16 / 4, x = 4

Приклад 5: 5x + 6 = 31, 5x = 31 - 6, x = 25 / 5, x = 5

На Головну

3. Множення та ділення алгебраїчних виразів

Теорія: Множення та ділення алгебраїчних виразів дозволяє спростити вирази за допомогою правил множення та ділення. Це включає використання властивостей дистрибутивності, скорочення дробів та розкриття дужок.

Приклад 1: (2x)(3y) = 6xy

Приклад 2: (4a)(5b) = 20ab

Приклад 3: (3x + 2)(x - 1) = 3x^2 - 3x + 2x - 2 = 3x^2 - x - 2

Приклад 4: (2x + 3)(x + 4) = 2x^2 + 8x + 3x + 12 = 2x^2 + 11x + 12

Приклад 5: (6x^2) / (3x) = 2x

4. Розв'язування систем лінійних рівнянь

Теорія: Система лінійних рівнянь — це набір рівнянь, які одночасно повинні виконуватися. Рішенням системи є таке значення змінних, при якому всі рівняння одночасно справджуються.

Приклад 1: 2x + 3y = 7, x - y = 1. Рішення: x = 2, y = 1

Приклад 2: x + y = 4, x - y = 2. Рішення: x = 3, y = 1

Приклад 3: 3x + 2y = 10, x - y = 1. Рішення: x = 4, y = 3

Приклад 4: 4x - y = 5, 2x + y = 8. Рішення: x = 3, y = 2

Приклад 5: x + 2y = 7, 3x - y = 8. Рішення: x = 3, y = 2

На Головну

5. Степені та корені

Теорія: Степінь числа — це вираз, який означає множення числа на себе певну кількість разів. Корінь числа — це число, яке при піднесенні до певного ступеня дає задане число.

Приклад 1: 2^3 = 2 × 2 × 2 = 8

Приклад 2: √9 = 3

Приклад 3: 5^2 = 25

Приклад 4: √16 = 4

Приклад 5: 3^4 = 81
